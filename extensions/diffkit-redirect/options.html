DiffKit

Redirect rules

Add only the GitHub routes DiffKit currently supports. A rule can match a single exact URL or a regex pattern over the full URL.

Rule shape

{
  "id": "github-pull-details",
  "label": "Pull request details",
  "description": "Redirect GitHub pull request detail pages to DiffKit.",
  "enabled": true,
  "match": {
    "urlRegex": "^https://github\\.com/([^/?#]+)/([^/?#]+)/pull/(\\d+)/?$",
    "excludeUrlRegexes": []
  },
  "redirect": {
    "replacement": "https://diff-kit.com/$1/$2/pull/$3"
  }
}

label is optional but recommended so the popup can show a clean toggle name. For one-off routes, use match.exactUrl and redirect.url instead.
